# MyCTiger

C code generation using the Ring programming language

Effectively turning Ring into a domain-specific language (DSL) for generating and executing C programs. 

Enjoy the performance and efficiency of the C language while harnessing the productivity of creating and using DSLs with Ring

## Features

* Use Ring source code at compile-time
* It turns Ring into a meta-language for C.
* This allows direct injection of raw C code into the output.
* Use any C source code and libraries (No binding is required)
